# Correcciones recurrentes

Este archivo sirve para registrar errores o correcciones que el usuario haga a
los agentes de codificacion de forma repetida.

Usalo para convertir feedback recurrente en reglas operativas.

## Formato sugerido

```markdown
## YYYY-MM-DD - <tema>

- Error observado:
- Correccion del usuario:
- Regla para proximas veces:
```

## Entradas

- _Sin correcciones registradas._
